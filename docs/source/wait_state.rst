Wait State
**********

The ``Wait`` class pauses the branch the state is defined within, until the specified date/time or interval has passed.

For more details on ``Wait``, see the `AWS documentation <http://docs.aws.amazon.com/step-functions/latest/dg/amazon-states-language-wait-state.html>`_.
