Fail State
**********

The ``Fail`` class acts as a end state of a state machine branch, signifying the state machine failed.

For more details on ``Fail``, see the `AWS documentation <http://docs.aws.amazon.com/step-functions/latest/dg/amazon-states-language-fail-state.html>`_.
