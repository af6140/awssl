Retrier
*******

The ``Retrier`` class defines how to respond to a failure in a ``Task_State`` or ``Parallel`` execution.

For more details on ``Retrier``, see the `AWS documentation <http://docs.aws.amazon.com/step-functions/latest/dg/amazon-states-language-errors.html#amazon-states-language-retrying-after-error>`_.
