Succeed State
*************

The ``Succeed`` class acts as a end state of a state machine branch, with the ability to provide filtered output data.

For more details on ``Succeed``, see the `AWS documentation <http://docs.aws.amazon.com/step-functions/latest/dg/amazon-states-language-succeed-state.html>`_.
