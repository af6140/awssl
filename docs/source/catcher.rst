Catcher
*******

The ``Catcher`` class defines how to continue processing when a failure occurs in a ``Task`` or ``Parallel`` execution.

For more details on ``Catcher``, see the `AWS documentation <http://docs.aws.amazon.com/step-functions/latest/dg/amazon-states-language-errors.html#amazon-states-language-retrying-after-error>`_.
